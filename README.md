# EntryPro

Система автоматического учета рабочего времени на основе геолокации. Приложение фиксирует нахождение сотрудника на объекте и передает данные на сервер в реальном времени.

## Суть проекта
Приложение использует Yandex Maps SDK для создания геозон. Когда сотрудник заходит в заданный радиус, смена открывается автоматически. При выходе из зоны смена закрывается. Все события логируются напрямую в базу данных PostgreSQL на удаленном сервере (VDS Ubuntu), что исключает возможность подделки времени в настройках телефона.

## Технический стек
* **Android:** Kotlin, Jetpack Compose.
* **Карты:** Yandex Maps SDK.
* **Фоновые задачи:** Foreground Service (для стабильной работы трекинга при свернутом приложении).
* **Бэкенд:** PostgreSQL, сервер на Ubuntu.
* **Архитектура:** MVVM.

## Что реализовано (v0.4.1)
* Автоматический старт и стоп смены по координатам.
* Система ролей: раздельный интерфейс для рабочего и директора.
* Защита от кратковременной потери GPS (10-минутный фильтр перед закрытием смены).
* Прямая синхронизация логов с сервером.

## В планах (v0.5.0)
* Детектор Fake GPS и запрет на использование эмуляторов.
* Выгрузка отчетов в формате Excel.
* Обновление визуальной части интерфейса.
